Contribute

The unspent.space project uses github.com for its documentation and any software packages that are forked or built.

You can support the project by:

  • submitting a pull request
  • liking unspent.space on nostr or twitter
  • join the conversation on our discord server
  • come and have a chat on the podcast
  • donate to unspent.space at unspent@getalby.com
